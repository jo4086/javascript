### Higher-Order Function (HOF) (고차 함수)

1. 함수를 인자로 받는 경우

2. 함수를 리턴하는 경우

    - Currying (커링 함수)
        - 하나의 인자를 받는 함수로 표현.
        - 해당 함수는 다른 함수를 리턴, 리턴 함수역시 하나의 인자를 받아야함
    - Partial Application (파셜 애플리케이션)
        - 함수가 받는 인자는 하나 이상일 수 있음
